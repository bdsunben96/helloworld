# Hello World 响应式布局示例

## 项目简介

这是一个简单的响应式网页布局示例项目，展示了如何使用HTML和CSS创建一个包含头部、内容区域和底部的基础网页结构。

## 功能特点

- 响应式布局设计，适应不同屏幕尺寸
- 包含头部导航菜单（Home、About、Contact）
- 三栏式内容区域（左侧边栏、主内容区、右侧边栏）
- 底部页脚区域
- 使用Flexbox进行布局控制

## 文件结构

- `index.html` - 主页面HTML文件，包含页面结构和内联CSS样式
- `README.md` - 项目说明文档

## 如何使用

1. 克隆此仓库到本地
2. 在浏览器中打开`index.html`文件
3. 可以根据需要修改HTML和CSS来自定义布局和样式
